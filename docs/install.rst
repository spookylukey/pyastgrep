============
Installation
============

Python 3.8+ required.

We recommend `pipx <https://pipxproject.github.io/pipx/>`_ to install it
conveniently in an isolated environment:

.. code-block:: shell

   pipx install pyastgrep


You can also use pip:

.. code-block:: shell

   pip install pyastgrep


Or:

.. code-block:: shell

   python -m pip install pyastgrep
